# net-snmp-tools

Tools to compile, install, update, and remove Net-SNMP on Windows systems.

---

## Compile-NetSNMP.cmd
### Purpose
Compile OpenSSL and Net-SNMP.
<br><br>

## Install-NetSNMP.ps1
### Purpose
Install or update Net-SNMP on a Windows system.
<br><br>

## Remove-NetSNMP.ps1
### Purpose
Uninstall Net-SNMP
